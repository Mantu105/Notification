Show Dynamic Notifications in Odoo 17

The Show Notification module lets you trigger dynamic toast-style notifications inside the Odoo backend interface using the OWL framework.

Features:

  • 🔥 Use native OWL NotificationService
  • 📦 Easy integration with any backend JS
  • 🚀 Minimal code, modern structure

How to Use:


import { showCustomNotification } from "@show_notification/notification";

showCustomNotification(env, "Your custom message here!");
